
# Future External Consumer Slots

AiNIR may later expose verified semantic intent artifacts to external consumers
such as compilers, planners, workflow engines, or runtimes.

This public demo does not implement any such consumer integration. The only
current rule is that future export artifacts must be derived from an AiNIR Trust
Gate decision with `status: passed`.

External consumers are out of scope for the current pre-v1 public demo.
